# 2025年终总结

![](./images/sysu2026.jpg)

**这一年还是遇到了挺多事，见证了挺多变化，经历了挺多成长的：**

1月：期末考后前往北京、天津旅游，10年后再游京城，感觉还是有所不同。也在天津感受到了独特的城市氛围。充实的一个月

2月：春运，第一次开近1000km的长途回老家，在老家期间也充当司机，以前坐过的路又开了一遍。奔驰的一个月

3月：主要是拍飞机，月初和朋友在白云机场小土坡拍747和919，月底时又用了一整个周末往返广深港三地拍机，算是把白云能落的机型几乎都拍到了。飞友的一个月

4月：大部分时间在准备转专业考试，二教晚自习成了每天的日常。希望与焦虑并行、坚定前行的一个月

5月：生日恰好在笔试的前两天，简单买了个小蛋糕平静地过完了19岁生日，所幸中旬的笔试相当顺利，21号的面试虽磕磕绊绊但影响不太大，当晚收到成功的消息，即使现在回忆还是热泪盈眶。“当幸福来敲门”的一个月

6月：翘了最后一节思政课和朋友去拍青铜剑，考完了最后一个相对轻松的期末考，月底于滨江东首发体验10、12号线。车迷的一个月

7月：月初前往香港拍摄山东舰，随后飞往成都，游览九寨沟与犍为，还和高中同学一同参观成电，人民公园。旅行的一个月

8月：与好友租车自驾潮汕，经历过堵在路上近4个小时，尝了当地人也不吃的生腌，体验了电竞酒店五连坐。欢笑的一个月

9月：月初参加了全国数模比赛，月中前往深圳东、平湖打印红票，月末拍摄并乘坐双ss8牵引的K6597前往深圳，告别纸质报销凭证。不舍的一个月

10月：国庆前往HKG与前辈一同拍摄A346，入坑了BangDream，陆陆续续把动画三季和pico追完了，同时平转的课业压力也开始具化，忙于应付作业，也焦虑着未来。内耗的一个月

11月：月中借着校运会的机会回到阔别近一年半的高中校园，又一次穿上了校服，于梦剧观看了我冠全程，也感慨于高中生们的活力与青春，为ID而自豪，月底又参加了漫协的全员大会，跟熟悉的伙伴们聚了几次餐。感动的一个月

12月：经历了最繁忙的第14周（一门期末一门期中、六级、1000m体测、实验验收、小测），期末临近，也愈感觉时间不够用了。忙碌的一个月

较于去年还有半年多时间在珠海，今年大部分时间都是在广州度过的。同去年一样，我仍在继续着对这座城市的探索，感受它的节奏。诚然下半年被流放小谷围岛，常居番禺，“进城”的次数大幅减少。且由于繁重的课业，现实与理想的落差逐渐显现，长期正反馈的缺乏，缺少投入于兴趣爱好的时间等都让我在计院的学习生活遇到了不小的阻碍。这一学期基本上是带着一堆debuff在生活学习，好在还是能坚持下来的。

去年的年终总结中，我写到将未来的轨迹交由明年的我来书写。现在的我已经以转专业书写下大学生涯的浓厚一笔，可下一步又该怎么走？今年的我没能找到一个使自己满意的答案，还因此内耗了相当一段时间。也许将路走下去，那个满意的答案自然会显现出来，那不妨就将这一任务交由明年的我吧。

祝新年快乐！
